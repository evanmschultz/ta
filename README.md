# ta

A tiny MCP server that lets LLM coding agents read and write TOML files as if they were a structured database — with schemas to keep agents honest.

See [`docs/ta.md`](docs/ta.md) for the design document and [`docs/PLAN.md`](docs/PLAN.md) for the MVP build plan.

## Status

Pre-alpha. Scaffolding in progress — see `docs/PLAN.md` phases.
